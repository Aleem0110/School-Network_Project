## 📖 Project Overview

This project demonstrates the design and implementation of a secure and scalable enterprise school network using Cisco Packet Tracer.

The network is divided into multiple VLANs with Inter-VLAN Routing, DHCP, NAT/PAT, ACL, SSH Remote Management, Wireless Access Point connectivity and Internet access.

The project follows real-world enterprise networking practices and is intended for learning, portfolio presentation and CCNA-level networking practice.
